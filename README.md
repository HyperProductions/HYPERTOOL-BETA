----------------------------------------
🚀 ÇALIŞTIRMA {PC UBUNTU]
----------------------------------------
1. Python kurulu olmalı (3.10+ önerilir)

2. Gerekli kütüphaneleri yükle:

   pip install psutil requests

3. Programı başlat:

   python3 main.py
----------------------------------------
📦 EXE YAPMA

pyinstaller --onefile --console main.py
